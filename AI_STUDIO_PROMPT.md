# ワイヤーフレームHTML生成プロンプト（AI Studio用）

このプロンプトをGoogle AI Studioにコピーして使用してください。

---

## プロンプト

```
あなたはWebデザイナーです。クライアントへの原稿依頼用のワイヤーフレームHTMLを作成してください。

【重要なルール】
このHTMLは自動ツールで解析され、Excelの原稿入力シートが生成されます。
以下のdata属性ルールを必ず守ってください。

## data属性の仕様

テキストが入る要素には必ず以下のdata属性を付けてください：

1. **data-section**（必須）: セクション名
   - 例: "ヒーロー", "プロフィール", "Q&A", "スケジュール", "フッター"

2. **data-label**（必須）: 要素名（15文字以内で簡潔に）
   - 良い例: "氏名", "キャッチコピー", "Q1タイトル"
   - 悪い例: "詳細：MVメイン写真プレースホルダー"（長すぎる）

3. **data-limit**（任意）: 文字数の目安
   - 例: data-limit="30"

## 記述例

```html
<!-- ヒーローセクション -->
<div class="hero" data-section="ヒーロー" data-label="メイン写真" data-limit="0">
  写真が入ります
</div>
<h1 data-section="ヒーロー" data-label="氏名" data-limit="20">山田 太郎</h1>
<p data-section="ヒーロー" data-label="キャッチコピー" data-limit="50">
  お客様の人生に寄り添う
</p>

<!-- Q&Aセクション -->
<h2 data-section="Q&A" data-label="Q1タイトル" data-limit="30">現在の仕事は？</h2>
<p data-section="Q&A" data-label="Q1回答" data-limit="200">
  営業として働いています...
</p>
```

## レイアウトのルール

1. **ヒーローセクションの高さは400px以下**に制限してください
   - `height: 400px;` または `max-height: 400px;`
   
2. **全体の幅は800px**を想定してください

3. プレースホルダー画像は背景色と枠線で表現してください
   ```css
   .placeholder {
     background: #e0e0e0;
     border: 2px dashed #999;
     display: flex;
     align-items: center;
     justify-content: center;
   }
   ```

4. **視認性を重視**：テキストは読みやすい大きさで

## 生成するページ

【ここにページの内容を記載】

例：
- 社員インタビューページ
- サービス紹介ページ
- 会社概要ページ
など
```

---

## 使い方

1. 上記のプロンプトをAI Studioにコピー
2. 「【ここにページの内容を記載】」の部分を具体的なページ内容に置き換え
3. 生成されたHTMLをダウンロードしてStreamlitアプリにアップロード

---

## サンプル：社員インタビューページの場合

```
【生成するページ】

社員インタビューページを作成してください。

必要なセクション：
1. ヒーロー（写真、氏名、所属、キャッチコピー）
2. プロフィール（導入文）
3. Q&A（4つの質問と回答）
4. スケジュール（1日の流れ、5〜7項目）
5. フッター（一覧に戻るボタン）

デザインテーマ：赤をメインカラーにしたシンプルなデザイン
```

---

## 注意事項

- **data-section**と**data-label**は必ず両方付ける
- **data-label**は15文字以内
- ヒーロー画像は**高さ400px以下**
- 日本語で記述する
